# 开发环境要求

  1. 基于 Ubuntu 或者 Debian，系统源改为国内源，
  2. 系统时间为东八区
  3. 安装 node、golang、python，源都改为国内源
  4. 使用 ssh 秘钥登录，方便 vscode 链接
  5. 生产目录映射到宿主机目录，防止工作数据丢失
  6. 暴露端口数至少 40 个，以满足折腾需求
  7. 解决中文乱码问题
