# 安装到移动硬盘的方式

在正常的 windows 系统中使用 rufus 工具写入镜像到移动硬盘

# rufus 工具设置

# 设置 rufus ，以显示移动硬盘

默认情况下 rufus 只显示 U 盘，要让其显示移动硬盘，需要在 显示高级设备选项 下勾选 显示 USB 外置硬盘

# 镜像选项 选择

选择 Window To Go

# 其他保持默认即可

# 修改引导方式

进入 BIOS ，将引导方式由 Legacy 改为 UEFI ,

# 从移动硬盘启动

重启系统， BIOS 将默认自动从移动硬盘启动进入系统

# 切换内置硬盘系统

只需要进入 BIOS ，将引导方式由 UEFI 改为 Legacy 即可，
