# tox1 刷 slimbox

工具使用 Amlogic USB_Burning Tool v3.1.0.zip (点击下载)

其他版本在此

1. 下载固件 (点击进入下载页面)，导入 UBT 中
2. 准备 USB 公对公线，一头先连接 tox1 ，用牙签顶住 tox1 的复位按钮，再将 USB 公对公线另一头连接电脑，等 UBT 上显示 连接成功 后送掉牙签。

刷入时记得将 擦除Bootloader 勾选去掉，否则不能刷入。
