# 二、nginx 配置 webUI 管理工具

均为 docker 容器部署

# nginx proxy manager

# 优点

可容器化部署，轻量化（两个容器占用共 71M+45M 内存）、无汉化

# 缺点

谈不上缺点的缺点 ---- 两个容器
https://nginxproxymanager.com/
部署：
新建 docker-compose.yml

然后在上述文件目录下执行： docker-compose up -d

如果已有数据库，则运行以下命令：

# nginx webui

# 优点

国人使用 JAVA 开发的，语言友善，只有一个容器，便于管理

# 缺点

占用资源很大（同样环境中该容器占用 205M 内存）
